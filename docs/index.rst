
Etch Documentation
==================

Etch is a project to manage bonuses by allows key functions as follows - 

* Add Bonuses/Rewards
* Approving payments
* Checking targets
* Listing Bonuses
* Buying Credits

The system works with wallets and therefore options exist to add wallets and assign them to bonuses.
